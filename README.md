# Whisk

A lightweight, user-friendly, fast **M4A downloader for Windows**
written in Python.

## Features

-   Smooth fade-in interface
-   Customizable download folder
-   Real-time progress updates
-   Built using **yt-dlp** for reliable downloading

## Installation

### 1. Install Python

Download Python 3.10+ from: https://www.python.org/downloads/

Make sure to check **"Add to PATH"** during installation.

### 2. Install Requirements

    pip install -r requirements.txt or do pip install yt-dlp

### 2. Extract Whisk

    Extract whisk on a empty folder

## Usage

Running the program:

    In the folder where you extracted the whisk, open command prompt and do main.py 

1.  Select a download folder\
2.  Paste a YouTube URL\
3.  Click **DOWNLOAD**

## Requirements

    yt-dlp
